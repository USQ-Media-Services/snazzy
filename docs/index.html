Easy Installation

To install and start using the icons, all you need to do is copy the following line of code and paste it at the top of the HTML of your course:

<script src="https://www.usq.edu.au/apps/multimedia/snazzy/snazzy.min.js"></script>

Then exit the HTML editor! Then, as you're writing your content, wherever you want an icon appear, just type the shortcodes below Want an animated cloud? Just type [{cloud}]!
